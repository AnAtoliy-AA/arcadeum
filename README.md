# aicoapp

Monorepo containing:

- `fe/`: Expo React Native app (mobile)
- `be/`: NestJS API server (backend)

Getting started and detailed setup live in subproject READMEs:

- Frontend: `fe/README.md`
- Backend: `be/README.md`

Sentry setup and CI guidance are documented per subproject:

- Mobile Sentry: see `fe/README.md` (uses `@sentry/react-native` and Expo tooling)
- Backend Sentry: see `be/README.md` (uses `@sentry/nestjs`)
